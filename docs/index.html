VisComm Helpers

A small set of browser tools to make marking, navigation and class admin in Canvas a bit less painful. Click a button below to install Tampermonkey.

🐵 Get Tampermonkey
How it works: install Tampermonkey first, then click the helper you want below. Tampermonkey should open the install screen automatically. Install the extension in your browser toolbar to easily turn the helpers on or off.
📊

Benchmarker

Sort students by grade band in a quick first pass. Then you can settle down and mark all your HDs in one go, know who is failing and tackle all the passes at once.

Install Benchmarker
🗂️

Tutorial Sorter

Import your course roster from Allocate+ and navigate Speedgrader by tutorial group. Easy way to organise your marking. This panel can export a .csv tutorial list for bulk import into Canvas so you can make real Canvas groups too!

Install Tutorial Sorter
🏃

When Will It End?

Track your marking progress and see how long it will take to mark your classes. Bonus Prince of Persia motivational animation

Install WWIE
🎚️

Rubric Slider

If you use grade bands in your rubric this slider is a quick and easy way to finetune that HD to an 85 or a 100.

Install Rubric Slider
📋

Copy Paster

Comment like a boss! Click to insert your professional opinion. Save your best comments to export for next time.

Install Copy Paster

Rubric Chooser

Build rubric criteria from the VisComm rubrics and export a Canvas-ready CSV to bulk import to Canvas. Never construct a criterion one at a time again.

Install Rubric Chooser
